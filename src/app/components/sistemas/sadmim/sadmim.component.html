El Sistema Integrado de Control Administrativo (SADMIM), ha sido desarrollado por profesionales con alta capacidad y amplia experiencia en la Administración Pública, que han aunado esfuerzos para la concepción de este importante sistema. En él hemos incorporado las normativas y controles exigidos por las diferentes leyes y entes gubernamentales que rigen y supervisan la buena labor de los Organismos Públicos. SADMIM, brinda un eficaz control y una amplia seguridad a todos los niveles de la administración, dotando a los usuarios de los niveles más operativos de una herramienta versátil para su operación y dotando a la Gerencia Administrativa de una amplia, oportuna y veraz información para la toma de decisiones. SADMIM, ha sido concebido bajo un esquema modular, donde cada uno de los nodos o Sub-Sistemas que lo conforman representan una gama de las actividades administrativas que las unidades de trabajo de los organismos deben realizar. Aun cuando su operatividad sea independiente, se mantienen interrelacionadas, permitiendo incorporar nuevas aplicaciones en la medida en que sea necesario. Integración Administrativa: constituye uno de los principales beneficios que otorga el Sistema SADMIM, la información generada por cualquiera de las unidades administrativas es compartida automáticamente con el resto de los módulos que constituyen el sistema, garantizando la homogeneidad y precisión en la información. SADMIM, es un sistema multiplataforma bajo Software Libre el cual puede operar con Servidores bajo plataforma Unix, Linux, Windows NT, Windows 98, Windows 95, Windows ME, Windows XP. Está desarrollado bajo un entorno RAD en el cual los usuarios pueden aprovechar las ventajas de un modelo centrado en la base de datos que crea separado HTML y código de. Servidor. La Base de Datos se encuentra en MySql Server y el código de servidor bajo PHP. Los profesionales de nuestra empresa garantizan la migración de la data de su sistema actual, lo cual permitirá a la administración municipal continuar de forma transparente sus procesos administrativos. Siempre y cuando la data cumpla con los requerimientos básicos de bases de datos.
